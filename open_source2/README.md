## 💻기능과 특징
> Fetch API & map()

Fetch API를 활용하여 외부 JSON 데이터를 로드함으로서, 필요한 리스트 항목만을 선별적으로 렌더링합니다. 이를 통해 사용자 집중도를 향상시키며, 확장성과 유지보수를 용이하게 할 수 있습니다.
